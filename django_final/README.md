# Django App

A basic survey Django App 

Prerequisites: Have [Python](https://www.python.org/downloads/) and [Django](https://docs.djangoproject.com/en/3.1/topics/install/#installing-official-release) installed

'python manage.py runserver' to run the app

Navigation:

/survey
/admin

(Note: for '/admin' must create superuser before running server.  Run 'python manage.py createsuperuser')
